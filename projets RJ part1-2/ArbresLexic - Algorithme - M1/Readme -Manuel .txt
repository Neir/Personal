Lancer le fichier Algav.jar via la commande : java -jar Algav.jar

Pour importer un dossier complet, par exemple l'oeuvre de Shakespeare, cliquer sur importer texte, selectionner le dossier
et cliquer sur ouvrir.

Pour importer le projet dans Eclipse, ne pas oublier de mettre l'image Algav.jpg dans le dossier du projet dans  Bin/AffichageGeneral/
